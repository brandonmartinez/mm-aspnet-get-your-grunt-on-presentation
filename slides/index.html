And why all the noise?

Grunt is a JavaScript Task Runner that runs on Node.js.

It can be used to minify and combine JavaScript, preprocess CSS via SASS or LESS, copy vendor assets from package manager like Bower and more.

A Whole New World

ASP.NET Core (previously 5/DNX/vNext/K) is a complete rewrite of ASP.NET.

Microsoft has made a huge effort in embracing the open source community, including utilities like Node.js, bower, and task runners like gulp/grunt.
